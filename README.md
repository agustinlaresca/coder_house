# Django Web Project with MVT Pattern

## Requisitos
- Python 3.x
- Django 4.x

## Instrucciones de configuraci√≥n
1. Clonar el repositorio:
   ```bash
   git clone <link-al-repositorio>
   cd myproject
   ```

2. Instalar las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

3. Aplicar las migraciones para configurar la base de datos:
   ```bash
   python manage.py migrate
   ```

4. Ejecutar el servidor de desarrollo:
   ```bash
   python manage.py runserver
   ```

## Funcionalidades
1. **Agregar un autor**: `/myapp/author/new/`
2. **Agregar un libro**: `/myapp/book/new/`
3. **Buscar libros**: `/myapp/search/books/`
